## Design Direction for Suggested Directions Page

The Suggested Directions Page should feel:

* Personalized
* Encouraging
* Beginner-friendly
* Flexible
* Clean
* Easy to compare

### Visual style

* White background
* Blue primary CTA
* Light blue active state
* Rounded role cards
* Skill tags
* Soft borders
* Simple illustrations
* Clear recommendation labels
* Consistent with previous onboarding pages
